# Friseur Beratungs App

### Beschreibung

Diese App führt Kunden vor Ort durch ein dialogbasiertes Frage-Antwort-System auf einem iPad, um eine gezielte
Produktempfehlung geben zu können

#### Produkte

Es geht hierbei hauptsächlich um neue Produktserien, um diese besser an den Kunden zu gewöhnen.
In diesem Fall die Clean Beauty Serie von Paul Mitchell.
